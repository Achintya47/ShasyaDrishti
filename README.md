## 📁 Repository Structure

The repository is organized to cleanly separate datasets from modeling workflows:

```
ShasyaDrishti/
│
├── Datasets/
│   ├── Price Forecasting/          # Historical commodity price data
│   ├── ShasyaDrishti AQI/          # Air quality datasets
│   ├── ShasyaDrishti Rainfall/     # Rainfall and climate data
│   └── ShasyaDrishti Yield/        # Yield-related datasets
│
└── Processes/
    ├── ShasyaDrishti AQI.rmp
    ├── ShasyaDrishti Price Forecasting.rmp
    ├── ShasyaDrishti Rainfall Predictor.rmp
    └── ShasyaDrishti Yield Predictor.rmp
```

| Folder | Description |
|---|---|
| `Datasets/` | All input data used across modeling pipelines |
| `Processes/` | Altair AI Studio workflow files (`.rmp`) |

---

## 🤖 Prediction Modules

| Module | Description |
|---|---|
| **AQI Predictor** | Forecasts air quality index based on environmental features |
| **Rainfall Predictor** | Models precipitation patterns using climate indicators |
| **Yield Predictor** | Estimates crop yield from soil, weather, and AQI data |
| **Price Forecasting** | Projects commodity prices using historical market data |
